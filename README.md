# New Pokemon Snap Locator
Application for viewing which pokemon are found on which islands.
# Run The Source Code
#### Download Python
https://www.python.org/downloads/
#### Verify Version Once Installed
py --version
#### From Your Download Directory Run The Source Code
py location.py
